# cpp 课程作业

## 协议

请注意，此项目使用 `GPL-3.0` 协议。

## 作业要求

- [x] 400 行
- [x] 5 个菜单功能模块
- [x] 有不同类、构造函数定义及使用、拷贝构造函数
- [x] 友元函数
- [x] 友元类
- [x] 多重继承
- [x] 虚基类
- [x] 虚函数
- [x] 抽象类
- [x] 运算符重载
- [x] 异常处理

## 如何使用

- 下载 [MinGW](https://osdn.net/projects/mingw/downloads/68260/mingw-get-setup.exe/)，勾选 `mingw32-gcc-g++-bin`
- 将它的 `bin/` 加入环境变量，如 `C:\MinGW\bin`
- 进入 `Fe_56/` 后执行 `g++ main.cc`
- 运行编译后生成的 `.exe` 文件
